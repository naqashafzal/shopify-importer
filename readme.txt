A simple plugin to import products from your Shopify store into WooCommerce.

== Description ==

This plugin provides a simple interface to connect to your Shopify store via its API and import your products into WooCommerce. It handles simple and variable products, including their names, descriptions, prices, and images.

== Installation ==

1. Upload the `shopify-importer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Shopify Importer' menu in your WordPress admin to configure the settings.

== Frequently Asked Questions ==

= Does this import product variations? =

Yes, it imports product variations as WooCommerce variable products.

= Does this import product images? =

Yes, it imports the main product image as the featured image in WooCommerce.
